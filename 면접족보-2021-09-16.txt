1. 메소드 오버로딩이란? 

한 클래스 내에 이미 사용하려는 이름과 같은 이름을 가진 메소드가 있더라고 매개변수의 개수 또는 타입이 다르면, 같은 이름을 사용해서 메소드를 정의 할수있는 것이 메소드 오버로딩이라고 한다.

2. 생성자 오버로딩이란?

한 클래스 내에 같은 이름의 메소드를 중복하여 정의하고, 클래스로부터 객체를 생성할 때 필요한 변수들만 적절히 초기화하기 위해 사용되는 것을 의미한다.

9. 사용자로부터 받은 문자열(영문으로)에서 자음과 모음 개수를 계산하는 프로그램을 작성하라.
(연속적이로 종료하시겠습니까 y 할때 까지 계속)

소스코드

import java.util.*;

class English{
	
	Scanner sc = new Scanner(System.in);
	
	private String str;
	private int collection = 0;
	private int consonant = 0;
	private char ch;
	private String check;
		
	public void show() {
		while(true) {
			System.out.println("영문 입력");
			this.str = sc.next();
			for(int i = 0; i < str.length(); i++) {
				char ch = str.charAt(i);
				switch(ch) {
				case 'a': case 'e': case 'i': case 'o':  case 'u': case 'A': case 'E': case 'I':  case 'O': case 'U':
					collection++;
					break;
				default:
					consonant++;
				}
			}
			System.out.println("모음 : " + collection);
			System.out.println("자음 : " + consonant);
			System.out.println("종료하시겠습니까? 종료 : y 계속 : n");
			check = sc.next();
			if(check.equals("y") || check.equals("Y")) {
				System.out.println("종료하였습니다");
				break;
			}
			else {
				collection = 0;
				consonant = 0;
				continue;
			}
		}
		
	}
}
public class LastExam01 {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		English eng = new English();
		eng.show();
	}

}

11.2와 100 사이에 있는 모든 소수(prime number)를 찾는 프로그램을 작성하라. 
(주어진 정수 k를 2부터 k-1까지의 숫자로 나누어서 나머지가 0인 것이 하나라도 있으면 소수가 아니다.)

소스코드

import java.util.*;

class PrimeNumber{
	
	Scanner sc = new Scanner(System.in);
	
	private int num;
	private int count = 0;
	
	public void show() {
		System.out.println("숫자 입력");
		this.num = sc.nextInt();
		System.out.println("2 부터 " + num + "까지의 소수는");
		for(int i = 2; i < num; i++) {
			if(num % i != 0) {
				count++;
				System.out.print(i + " ");
				
			}
		}
		System.out.println("2 부터 " + num + "까지의 소수 갯수 : " + count);
	}
}
public class LastExam03 {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		PrimeNumber prime = new PrimeNumber();
		prime.show();
	}

}
