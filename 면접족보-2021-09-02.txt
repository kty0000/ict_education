1. 상수란 무엇이며 변수와의 차이는?
상수란 변수의 반대로 변하지 않는 값을 의미 합니다. (final)

2. 초기화란?
객체를 선언하고 값을 '최초'로 할당하는 것이다.

3. 리터럴이란?
자료형을 기반으로 표현이 되는 상수를 의미한다.

4.아래의 내용이 에러가 나는 이유와 수정 방법은?

java
System.out.println(3147483647 + 3147483648); //에러남

에러가 나는 이유는 기본적으로 자바에서 정수 연산을 할 때 int형으로 연산하는데 int형의 최대값을 초과해서 에러가 발생함.

수정

System.out.println(3147483647L + 3147483648L);

5. 자동 형변환과 명시적 형변환에 대하여 설명하시오.
자동 형변환
작은 메모리 크기의 데이터 타입을 큰 메모리 크기의 데이터 타입으로 변환하는 행위를 말한다.

명시적 형변환
자동 형변환 규칙에 부합하지는 않지만, 형 변환이 필요한 상황이면 명시적 형변환을 진행한다.

6.아래의 에러가 나는 이유를 설명하고 수정 하시오.

int a = 3;
int b = 4;
	
double result = a / b;

System.out.println(result);

에러가 나는 이유는 int 연산 결과는 int인데(소수점을 날림) 그 값을 result으로 대입 하면서 double로 대입해서 .0이 붙게 된다.

수정

int a = 3;
int b = 4;
	
double result = (double)a / b;

System.out.println(result);

7. 이스케이프 시퀀스의 종류를 나열하고 기능을 설명하시오.
\b = 백스페이스 문자 : 키보드 backspace 기능
\t = 탭 문자 : 일정한 간격 띄우기
\\ = 백슬래시 문자 : 역슬래시( \ ) 문자 출력
\' = 작은 따옴표 문자 : 작은 따옴표( ' ) 문자 출력
\" = 큰 따옴표 문자 : 큰 따옴표( " ) 문자 출력
\n = 개 행 문자 : 줄 바꿈, 개행 기능
\r = 캐리지 리턴(carriage return)문자 : 동일한 줄의 맨 앞 칸으로 커서 이동

8. 아래의 출력 결과를 확인하고, 130이 나오는 이유를 설명


System.out.println('A'); 
System.out.println('A' + 'A'); 

출력 결과
A
130

130이 나오는 이유는 알파벳 A의 아스키코드 값은 65이기 때문에 연산하면 130이 나옴.

9. 결합 방향과 우선순위에 대하여 설명하시오.
우선순위 : 기본적으로 연산자에는 우선순위가 있으며, 괄호의 우선순위가 제일 높고, 산술 > 비교 > 논리 > 대입의 순서이며, 단항 > 이항 > 삼항의 순서임.
결함 방향: 연산자의 연산 진행방향은 왼쪽에서 오른쪽으로 수행되며, 단항 연산자와 대입 연산자의 경우에는 오른쪽에서 왼쪽으로 수행됨.

10. "2의 배수 또는 3의 배수?" 를 코딩으로 표현하시오.

11. && 와 || 설명하시오
&& 와 || 둘다 논리 연산자로,

&& : 모두 참(True)인 경우만 참(True) 나머지는 거짓(False)

|| : 모두 참(True)이거나 둘중 하나만 참(True)이여도 참

14. ASCII(아스키 코드)란?
American Standard Code for Information Interchange 의 약어 입니다. 
이는 미국 ANSI 에서 만든 표준 코드 체계 입니다. 주로 통신용 코드에 사용됩니다.
